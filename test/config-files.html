Testing Configuration File Presence

You will only see this page if one or more of the configuration files have not been renamed. You must edit each of the following files:

  • .htaccess
  • configuration.php-dist
  • assets/includes/custom-css.htm-dist
  • assets/includes/custom-js.htm-dist
  • assets/includes/footer.php-dist
  • assets/includes/header.php-dist

Please customize each file, even if just with valid ’blank’ code, then rename it by “Saving As...” and removing the “-dist” string from the extension.

After accomplishing this work, revisit the main page to be sure it worked.
